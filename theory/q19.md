## Q19: analogWrite() vs analogRead() — PWM in Arduino

### analogRead()
- **What it does:** Reads a voltage from an analog input pin and converts it to a number.
- **Range:** Returns a value between **0 to 1023** (10-bit ADC).
- **Pins:** Only works on analog pins — A0, A1, A2, A3, A4, A5.
- **Direction:** INPUT — Arduino is *reading* something from the outside world.

**Syntax:**
```cpp
int value = analogRead(A0);
```

**Practical IoT Example:**
Reading a soil moisture sensor. The sensor outputs a voltage depending on how wet the soil is. Arduino reads that voltage using `analogRead()` and decides whether to turn on a water pump.

---

### analogWrite()
- **What it does:** Sends a PWM (simulated analog) signal out of a digital pin.
- **Range:** Takes a value between **0 to 255** (8-bit).
- **Pins:** Only works on PWM-capable pins — marked with **~** on Arduino (3, 5, 6, 9, 10, 11).
- **Direction:** OUTPUT — Arduino is *sending* a signal to control something.

**Syntax:**
```cpp
analogWrite(9, 128);  // 50% duty cycle
```

**Practical IoT Example:**
Controlling the brightness of a status LED in a smart home system. `analogWrite(9, 255)` = full brightness, `analogWrite(9, 50)` = dim glow for night mode.

---

### What is PWM?

PWM stands for **Pulse Width Modulation**.

Arduino's digital pins can only output HIGH (5V) or LOW (0V). They cannot output 2.5V directly. PWM tricks the component by **rapidly switching between HIGH and LOW** many times per second.

The **duty cycle** controls how long the pin stays HIGH vs LOW in each cycle:

| Duty Cycle | analogWrite value | Effect on LED |
|---|---|---|
| 0% | 0 | Fully OFF |
| 25% | 64 | Dim |
| 50% | 128 | Half brightness |
| 75% | 191 | Bright |
| 100% | 255 | Fully ON |

Because this switching happens ~490–980 times per second, the human eye (or motor/LED) sees it as a steady intermediate value — not flickering.

### Why is PWM used in IoT?
- **Saves power** — instead of always full power, you control exact output level
- **Controls speed** — DC motors, fans
- **Controls brightness** — LEDs, displays
- **Controls position** — servo motors (used in robotics, smart locks)

### Quick Comparison Table

| Feature | analogRead() | analogWrite() |
|---|---|---|
| Direction | Input | Output |
| Pin type | Analog pins (A0–A5) | PWM pins (~3,5,6,9,10,11) |
| Value range | 0 – 1023 | 0 – 255 |
| What it does | Reads real voltage | Outputs PWM signal |
